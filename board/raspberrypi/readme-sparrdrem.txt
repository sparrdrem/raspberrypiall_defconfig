raspberrypiall_defconfig is a modified script that builds an ARMMP system for the Raspberry Pi. The following files have originated from sparrdrem/raspberrypiall_defconfig:

- board/raspberrypi/genimage-raspberrypiall.cfg
- board/raspberrypi/genimage-raspberrypilegacy.cfg
- board/raspberrypi/genimage-raspberrypimodern.cfg
- board/raspberrypi/readme-sparrdrem.txt
- board/raspberrypiall.ln
- board/raspberrypilegacy.ln
- board/raspberrypimodern.ln
- config/raspberrypiall_defconfig
- config/raspberrypilegacy_defconfig
- config/raspberrypimodern_defconfig

This is subject to change.
